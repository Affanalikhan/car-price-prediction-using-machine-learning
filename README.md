## 🎯 Overview

This project predicts the selling price of used cars based on features like brand, year, fuel type, transmission, mileage, and engine specs. It demonstrates a complete ML pipeline from data cleaning to model deployment-ready predictions.

---

## 🔍 What's Inside

- **EDA** — Price distribution, brand-wise analysis, feature correlations
- **Preprocessing** — Encoding categorical variables, handling nulls, outlier removal
- **Models** — Linear Regression, Random Forest, Gradient Boosting
- **Evaluation** — R², RMSE, feature importance visualization

---

## 📊 Key Features Analyzed

| Feature | Type | Impact |
|---|---|---|
| Year / Age | Numeric | High — newer = higher price |
| Fuel Type | Categorical | Medium — diesel tends higher |
| Transmission | Categorical | Medium — automatic premium |
| Kilometers Driven | Numeric | High — inverse relationship |
| Engine CC | Numeric | Medium — larger = pricier |
